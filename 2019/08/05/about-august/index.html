八月小记

8月 05 2019

好的，确认了一下我还没有忘记 Markdown 语法。但是我想这个名字我居然想了两分钟，看来还是陌生了呀。没说成 Fucked up 算好的。

从六月的期末考试，到七八月的辅导班以及参加国赛，这阵子事情比较多比较忙，但是我可说的真的太多太多了。我想想应该从哪开始说，让我按「事件发展顺序」一件一件说吧。😎

六月紧张的期末考试之余，还紧张开展着暑假的业务——辅导班做打算，一边做着物料，一边做着清晨和大热天去外面发着传单。一边剥削着自己，一边憧憬美好的未来。

大概七月多一点，我的主力信息流 App 即刻凉了，但是手机不能不刷。于是我转战了知乎和豆瓣（知乎真是鱼龙混杂，越刷越感觉 Low），在豆瓣我看见了这样一段话：

在现在短视频当道的时代，还在坚持文字写作。就像是在现代战场使用冷兵器，酷是酷，但是很没用，总之很酷。

我不太记得原文了，大概是这个意思。

在碎片阅读和快餐游戏、短视频的环境浸泡之下，我这把兵器钝化的程度非常厉害，前几年我还能写点小短篇、偶尔无病呻吟一下，现在已经没什么内容可以挤了。但是我仍旧乐意挥舞它，酷就完事了。

为了参加国赛，我们公司特意全体休假两天，没办法，谁让我是董事呢。

感觉有点喜欢上长春这座城市了，城市小而清净，从司机都食堂大妈都很让人感觉很亲近。可能是在赵本山的作品之下，对东北话产生后天的亲近感。

往返一整天的火车没有给我烦闷的感觉，恼火的是回来的时候，我的身份证连同车票一起丢了，好在出站时没有查票。后来我只好在网上买了张假票去应付报销，🤐没办法，车票太贵了我自己负担不起。

为什么不想找女朋友？因为我自私。

越发发现我是一个独居动物。为什么，因为独居太爽了了，请原谅我用这么肤浅的词来形容，可是独居就是太爽了。爽到这中间甚至容不下一个恋人。

我忍受不了太多事了，我忍不了冷落，忍不了流量明星，忍不了国产综艺。

本来想在这篇里面记录一下新发现的有趣的东西。不知道为什么和上一篇一样非常雷同地开始谈论恋爱的问题了大概是因为，上一次时间是5月20，这一次是七月初七吧。

简单一点儿吧，我爱《歌剧魅影》。

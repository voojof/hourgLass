我为什么来这——感谢这个时代

1月 24 2019

我其实在很多地方都留下过足迹，像是 Facebook、微博、微信、QQ 空间等等等等。但是我都多多少少有些不喜欢那里。

一个是因为它们中间有些越来越大众化，伴随者它公开的性质，那这代表着你可能会看到很多你不想看到的东西出现在你的主页。

而且在这些地方发东西我总有一个毛病，就是会过分地关注他人的看法，例如看看谁点了赞呀，看看谁又评论了什么呀。这使我受到了比较大的困扰，以至于我后来干脆关闭了入口，让我只能写和发送，看不到相关的回复。

亦或者是周围充满了认识你的人，我时常想说些闲言碎语，或者说的有些话又充满攻击性。我并不想让这些东西出现在别人的时间线上。然后我又疲于去配置可见不可见的权限，更加疲于每次发送东西都要反复斟酌看看有没有不妥的地方。

总的来说，就是不想看见某些人，又不想被某些人看见。

我也尝试过去一些小众的平台，要么就是发现其中有些功能不是我想要的，要么就是平台主流用户低龄化我有点受不了，要么就是这个平台太小众到让我担心它某一天就会消失……

总之，就没有让我满意的，后来我干脆不写东西了。

感谢这个时代。

这个时代有了不起的开发者，为互联网各个角落添砖加瓦，并且毫不吝啬自己的智慧，这些东西不仅他人免费使用，还可以在原来的基础上随意修改、发布。

这个时代有了不起的公司，Github 免费提供无限的网站容量，并且让用户做以任何用途。

这个博客就是基于 Hexo，修改后的 Polarbear 主题。服务来自 Github Pages。感谢他们。

我为什么不用诸如 WordPress 之类的工具搭建动态博客呢，主要是因为 Github 不支持，因为当时我当时看到 Hexo 的我就喜欢得不得了，漂亮、稳定，想怎么折腾怎么折腾（虽然现在有些东西我仍然没有配置完善😂），轻松的网站转移和备份。而且文章都是以 Markdown 形式单独保存，这一点简直深得我心。

而且搭在自己的网站上，这总没什么人看了，你想看你能找到你就看，你不想看我也没有逼你看了吧。

可以说是非常地舒服了。
